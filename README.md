# AI-Fitness-Tracker

##  Description
Application de Computer Vision qui détecte les postures et compte les répétitions d'exercices physiques via la webcam.

##  Stack Technique
- **Langage :** Python
- **Bibliothèques :** OpenCV, MediaPipe, NumPy

##  Fonctionnalités
- Détection de pose en temps réel
- Calcul automatique des angles articulaires
- Compteur de répétitions (Squats, Push-ups)
- Interface de feedback visuel
